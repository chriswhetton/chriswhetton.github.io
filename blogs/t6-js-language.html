JavaScript Language

JavaScript tells the browser how to change the web page in response to events that happen, HTML marks the content up into different structural types, like paragraphs, blocks, lists, images etc. CSS tells the browser how each type of element should be displayed depending on the media type being used.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'

Control flow is the order (sequencing) of things to occur to get the desired behaviour, whereas loops perform the same task over and over again until a set of criteria are met. Taking the 'brushing' your teeth analogy at a high level - you normally make your way to the bathroon, then open open the vanity removing the dental floss, after flossing you then take the toothbrush and toothpaste out, apply the toothpaste to the brush, then brush. Looping would be repeated brushing until the teeth are clean.

Explain the difference between accessing data from arrays and object literals.

Object literal values are accessed using dot notation (console.log(person.location);), Array values are accesssed using numerical index (console.log(array[2]);).

Explain what functions are and why they are useful.

A function is a block of code designed to perform a particular task; very useful when you need to use the same code many times, you only need to define it once.
