JavaScript Language - Explanations to a non-tech friend

parentheses () group operations together and take precedence. They are also used to store expressions and parameters when defining functions.

brackets [] are used to define and retrieve elements from an array.

braces {} are used for grouping of statements e.g. if multiple statements are required to be run in a loop.

single quotes ''and double quotes "" are used to define the data type string which represent text. Single and double quotes are interchangable, as long as the quotes at the start and end of the string match.
