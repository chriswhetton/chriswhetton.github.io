What do the functions .map(), .filter(), and .reduce() do? .map() basically creates a new array out of the results returned from an existing array. Applying .filter() to an existing array of elements will create a new array with only the filtered elements. The .reduce() function can be used to take a collection of numbers or a string of words and append or add them to a variable that you have declared.
