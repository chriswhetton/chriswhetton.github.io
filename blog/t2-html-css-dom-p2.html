Grid Based Design

Grid based design is a way of keeping elements aligned and ordered. By breaking a page into columns and rows, the grid system speeds up and improves design time, acting as a guide for placement, position, and scale of elements.


Responsive Webpages

Responsive webpages adjust the way content is laid out on a screen based on the screen size and capabilities of the device being used. For example, a user of a mobile phone would see content shown in a single column view to that of a user with a tablet, which might display the same content in two columns.


Semantic Structure

Semantic structure is the use of HTML markup; only selecting tags appropriate to the meaning of the content. Presentation and styling is the domain of CSS.
